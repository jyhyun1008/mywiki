# 75% LUCK!

## 화성

- `F major key`: 
    - 멜로디 구성음 중에 F가 압도적으로 많음. D는 잘 쓰이지도 않고 패싱 노트나 중요하지 않은 음에 쓰이는 반면 F는 중요한 자리나 목적 음으로 많이 쓰임.
    - 코드에서 Dm가 잘 쓰이지 않음. F major 키의 IV, V7이 D minor 키의 iv, V7보다 압도적으로 많이 쓰임. - 중요할 지는 모르겠음.

- M7, m7, 7 잘 안 씀 : 
    - 가볍고 담백한 분위기, 쨍하게 밝은 분위기. 간혹 도미넌트에서 7음을 칠 때 불안감을 더 강조하기 위해서

### 인트로

`F - C - Bb - F - Bb - C`
- I(M7) - V(7) - IV(M7) - I(M7), 해결 - IV(M7) - V(7)

`Bb(2박) - A7(2박) - Dm(1박) - C#dim(1박) - C7(1박) - Bb(1박)`
- IV(M7) - V7|VI - VIm(7) - bVIdim7, 패싱 - 패싱 - IV(M7)

### 간주, 브리지

`Bb - Am - Bb - C7`
- IV(M7) - IIIm(7) - IV(M7) - V7

### 벌스

`F - Am - Bb - C - F - F7 - Bb - C`
- I(M7) - IIIm(7) - IV(M7) - V(7) - I(M7) - V7|IV - IV(M7) - V(7)

2절 벌스

`F - Adim/D# - Bb - C - F - F7 - Bb - C`
- I(M7) - IIIdim7 - IV(M7) 반음 위 - V(7) - I(M7) - V7|IV - IV(M7) - V(7)

### 프리코러스

`Bb - C - F - A7 - Dm - Bb - C - Bb - C`
- IV(M7) - V(7) - I(M7) - V7|VI - VIm(7) - IV(M7) - V(7) - IV(M7) - V(7): 4도 5도 반복

### 코러스

`F - C - Bb - F - Bb - F - Bb - C7`
- I(M7) - V(7) - IV(M7) - I(M7), 해결 - IV(M7) - I(M7) - IV(M7) - V(7)

## 트랙

### 음 높이

- 베이스 → 피아노 왼손 → 패드 왼손 → 백킹기타 → 피아노, 패드 오른손 → 기타 → 리드 → 아르페지오

### 패닝

- 백킹기타 L ← 아르페지오 ← 기타 ← 피아노 ← 킥, 베이스 → 보컬 → 리드 → 패드 → 백킹기타 R

### 살리고 싶은 트랙

- 보컬
- 리드
- 피아노

### 질감만 남기고 뒤로 빼고싶은 트랙

- 아르페지오
- 패드
- 기타들 (질감만 남기기)

### 제가생각하는 문제점

- 따뜻한 성질의 음색이 너무 많아요
    - 따뜻한 곡을 목표로 하긴 했어요
    - 톤 깎인 클린톤을 너무 선호하는 것 같아요
    - 근데 아르페지오도 사인파를 선호하는 것 같아요
    - 리드는 톱니랑 삼각파(옥타브올림) 합쳤는데 삼각파의 성질이 더 부각된 것 같아요
    - 피아노 톤 노브 올렸어요
    - 기타 앰프에서 Bright 스위치 켰어요
    - 일단 이대로가 베스트다 싶은 거 제외하고는 다들 만졌어요 (베이스, 아르페지오, 리드를 제외함)
